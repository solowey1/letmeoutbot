```bash
sudo bash install.sh
```

### Quick Development Setup
```bash
./quick-start.sh
```

### Manual Installation
See [Installation Guide](INSTALLATION.md)

## Database Options

- **SQLite** - Simple, file-based (testing)
- **PostgreSQL** - Robust, scalable (medium projects)
- **Supabase** - Managed cloud database (recommended for production)

## Support

- Check setup: `./check-setup.sh`
- View logs: `docker-compose logs -f`
- Read docs: Start with [Quick Start](QUICK_START.md)
