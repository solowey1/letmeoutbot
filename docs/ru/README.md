1. **[Быстрый старт](QUICK_START.md)** - Запустите за 5 минут
2. **[Руководство по установке](INSTALLATION.md)** - Детальные инструкции по установке

## Методы установки

### Автоматическая (Рекомендуется)
```bash
sudo bash install.sh
```

### Быстрая настройка для разработки
```bash
./quick-start.sh
```

### Ручная установка
См. [Руководство по установке](INSTALLATION.md)

## Варианты баз данных

- **SQLite** - Простая, файловая (тестирование)
- **PostgreSQL** - Надёжная, масштабируемая (средние проекты)
- **Supabase** - Управляемая облачная БД (рекомендуется для продакшн)

## Поддержка

- Проверить настройку: `./check-setup.sh`
- Посмотреть логи: `docker-compose logs -f`
- Читать документацию: Начните с [Быстрого старта](QUICK_START.md)
