# Краткое руководство

## Для продакшн (одна команда)

```bash
sudo bash install.sh
```

Ответьте на вопросы → Готово! ☕

---

## Для разработки

```bash
./quick-start.sh
```

Интерактивная настройка → Запуск в режиме разработки

---

## Что вам понадобится

### 1. Токен Telegram бота
- Откройте [@BotFather](https://t.me/botfather)
- Отправьте `/newbot`
- Включите платежи: `/mybots` → Выберите бота → Payments
- Скопируйте токен

### 2. Ваш Telegram ID
- Отправьте `/start` боту [@userinfobot](https://t.me/userinfobot)
- Скопируйте ваш ID

### 3. Outline VPN API URL
- Установите [Outline Manager](https://getoutline.org/get-started/)
- Создайте сервер
- Скопируйте "Management API URL" из настроек

### 4. База данных (выберите одну)

**Вариант 1: Supabase** (Рекомендуется)
- Перейдите на [Supabase](https://app.supabase.com)
- Создайте проект
- Получите URL и API ключ из Settings → API

**Вариант 2: PostgreSQL**
- Используйте существующий PostgreSQL сервер
- ИЛИ позвольте install.sh настроить локальный PostgreSQL в Docker

**Вариант 3: SQLite**
- Настройка не нужна
- Подходит только для тестирования

## Процесс установки

```
Запустить install.sh
  ↓
Выбрать базу данных (1-3)
  ↓
Ввести учётные данные БД
  ↓
Ввести конфигурацию бота
  ↓
Дождаться автоматической настройки
  ↓
Готово! ✅
```
